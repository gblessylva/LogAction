== Description ==

**LogAction - Activity Logs for Admin** is a powerful WordPress plugin that provides a detailed activity logging system for your website. It tracks user actions, changes, and events, allowing site administrators to monitor activity, enhance security, and improve user experience. With an intuitive interface and customizable features, LogAction makes managing your site's activity a breeze.




### Key Features:
- Tracks user logins, post updates, plugin activations, and more.
- View logs in an intuitive interface.
- Export logs to CSV for offline analysis or reporting.
- Customize logging preferences to suit your site's needs.
- Filter logs by date and actions.



This plugin is essential for administrators aiming to monitor activity, troubleshoot issues, and enhance their site's security posture.




== Installation ==

1. Upload the `logaction` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the **LogAction** menu in the WordPress admin panel to begin tracking activities.

== Frequently Asked Questions ==

= **Q: How do I access the logs?** 
**A:** After activation, navigate to **LogAction > Logs** in your WordPress admin dashboard.




= What activities does this plugin log? =  
The plugin logs actions like user logins, post and page edits, media uploads, plugin activations, and more.





= Can I export the logs? =  
Yes, logs can be exported as CSV files for easy offline analysis.




= How do I export the logs?**
Click the **Export** button on the logs page to download the logs in CSV format.




= Who can view the activity logs? =  
Only users with administrative privileges can view and manage logs.




= Is there a way to clear the logs?
Yes, on the settings page you can delete all the logded activities.
Also, You can select multiple logs to delete on the Logs page.
